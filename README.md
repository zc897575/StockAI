# 📈 AI股票分析系统
基于AkShare和Tushare的A股行情分析、模拟炒股、量化分析一体化Web系统

## ✨ 功能特性
### 1. 实时行情
- A股全市场实时行情展示
- 支持按股票代码/名称搜索
- 市场概览统计（涨跌家数、总成交额等）
- 支持按涨跌幅、成交量等排序

### 2. 股票深度分析
- 日/周/月K线图展示，支持均线叠加
- 自动计算常用技术指标（RSI、MACD、布林带、ATR等）
- 基本面财报数据查询
- 交互式图表，支持缩放、平移

### 3. 每日AI选股推荐
- 每日自动从沪深300成分股中筛选优质标的
- 基于量价配合策略，筛选上升通道股票
- 展示5日涨幅、当前价格等关键指标
- 推荐结果自动缓存，避免重复计算

### 4. 模拟炒股系统
- 用户注册/登录功能
- 初始资金10万元，完全模拟真实交易
- 支持买入/卖出操作
- 持仓管理、交易历史查询
- 自动计算持仓收益、收益率

### 5. 量化分析工具
- 双均线策略回测功能
- 自定义短/长期均线周期
- 自动计算策略收益率、最大回撤、胜率等指标
- 策略收益与大盘收益对比可视化

## 🛠️ 技术栈
- **后端/UI框架**：Streamlit
- **数据源**：AkShare、Tushare
- **数据处理**：Pandas、Numpy
- **技术指标**：TA-Lib
- **可视化**：Plotly
- **数据存储**：SQLite3

## 🚀 快速部署
### 环境要求
- Python 3.8+
- 内存 ≥ 2GB
- 网络可访问公开金融数据源

### 安装步骤
1. 克隆项目
```bash
git clone https://github.com/your-username/stock-analysis-webui.git
cd stock-analysis-webui
```

2. 安装依赖
```bash
pip install -r requirements.txt
```

3. 启动服务
```bash
streamlit run app.py --server.port 8501
```

4. 访问系统
打开浏览器访问：http://localhost:8501

### 云服务器部署
```bash
# 后台运行
nohup streamlit run app.py --server.port 8501 > app.log 2>&1 &

# 查看日志
tail -f app.log
```

## 📱 功能使用说明
### 实时行情
- 打开系统默认显示全部A股实时行情
- 顶部搜索框可输入股票代码或名称快速搜索
- 可选择按涨跌幅、成交量、成交额等排序
- 红色表示上涨，绿色表示下跌

### 股票分析
1. 输入要查询的股票代码（如000001）
2. 选择K线周期（日线/周线/月线）
3. 选择要查看的历史天数（30-365天）
4. 点击"开始分析"即可查看完整分析报告

### 每日推荐
- 系统每日自动生成5只精选股票推荐
- 推荐逻辑：量价配合、上升通道、基本面良好
- 点击股票代码可直接跳转至分析页面

### 模拟炒股
1. 首次使用需要注册账户，初始资金10万元
2. 登录后可查看账户总资产、可用资金、持仓市值
3. 买入：输入股票代码和数量，确认即可买入
4. 卖出：选择持仓股票，输入卖出数量即可卖出
5. 可在"我的持仓"查看当前持仓和收益情况
6. 可在"交易历史"查看历史交易记录

### 量化分析
1. 输入要回测的股票代码
2. 自定义短期和长期均线周期
3. 选择回测的历史时间长度
4. 点击"开始回测"即可查看策略表现和收益曲线
